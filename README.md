# Jarvis AI Assistant

An AI-powered virtual assistant built using Python with automation, voice recognition, GUI interface, and smart assistant capabilities.

## Features
- Voice Recognition
- AI Chat Responses
- Automation Tasks
- GUI Interface
- Speech Recognition
- OpenCV Integration
- Smart Assistant Functions

## Tech Stack
- Python
- Flask
- OpenCV
- SpeechRecognition
- HTML
- CSS

## Project Structure
- Backend
- Frontend
- Data
- Graphics

## Installation

```bash
pip install -r Requirements.txt
python Main.py
```
